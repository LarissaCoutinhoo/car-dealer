# Car Dealer - Relatório de Performance com Spring Batch

## Sobre o projeto

Aplicação desenvolvida para automatizar o processamento de relatórios de vendas enviados por diferentes filiais, consolidando as informações em um único relatório de performance.

## Objetivo do projeto

Automatizar a leitura e o processamento de múltiplos arquivos de vendas, reduzindo atividades manuais realizadas em planilhas e facilitando o acompanhamento de resultados por parte da matriz.
